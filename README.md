## рҹҢҚ What is Cosmeon?

Cosmeon is a **production-grade Geospatial Intelligence Platform** that transforms raw multi-modal satellite imagery into actionable risk assessments in real time. It's built for analysts, planners, and researchers who need to go beyond static maps and into dynamic, AI-driven geospatial insights.

The platform is anchored around three core **Analysis Orbs**:

| Orb | Focus | Data Source | Color |
|-----|-------|-------------|-------|
| рҹҢҠ **Flood Risk** | Inundation extent via SAR backscatter + GloFAS discharge | Sentinel-1 + GloFAS v4 | Cyan |
| рҹҸ—пёҸ **Infrastructure Exposure** | Water-human structure intersection | Sentinel-1 + OSM | Orange |
| рҹҢҝ **Vegetation Health** | NDVI anomaly + ERA5 dryness stress index | Sentinel-2 + ERA5 | Green |

## вңЁ Key Features

### рҹӨ– Machine Learning
- **XGBoost + LightGBM 55/45 Soft-Voting Ensemble** trained on 13 weather and terrain features вҖ” precipitation (7-day, 30-day), soil moisture, elevation, slope, GloFAS discharge anomaly, seasonal indicators, and more
- **Thread-Safe Concurrent Training** вҖ” `threading.Lock` prevents race conditions when multiple requests trigger simultaneous re-training; non-blocking acquire with graceful fallback
- **Deterministic Predictions** вҖ” local `np.random.RandomState(42)` per training run; eliminates global seed mutation so predictions are stable across concurrent requests
- **Background Pre-Training at Startup** вҖ” ML model warms up in a daemon thread before the first request hits, eliminating Render's 30s cold-start timeout
- **SHAP Explainability** вҖ” every prediction comes with a full feature importance breakdown so analysts know *why* a risk level was assigned
- **6-Month Forecasting** via ARIMA-style time-series extrapolation with confidence bands
- **Anomaly Detection** using Isolation Forest for detecting out-of-distribution satellite events

### рҹ“Ў Data & Fusion
- **Live GloFAS v4 Integration** вҖ” real river discharge data ingested per region; used as the ground-truth label source for ML training and as a live validation signal alongside the model
- **ERA5 Climate Reanalysis** вҖ” Open-Meteo API delivers hourly precipitation, ETвӮҖ, and soil moisture; aggregated into 7-day and 30-day windows per location
- **Vegetation Stress Index** вҖ” computed as a climatological dryness anomaly: `50 + (clim_mean_precip вҲ’ actual_precip) / clim_mean_precip Г— 50`, centred at 50 (normal), rising when drier than seasonal average, falling in wet conditions вҖ” eliminates the flat-zero bug that afflicted monsoon regions
- **Multi-Sensor Fusion** вҖ” dynamically weights SAR vs Optical imagery based on real-time cloud cover conditions (Sentinel-1 dominates when cloud cover > 70%)
- **Automated Change Detection** вҖ” calculates temporal deltas between baseline and current state, quantifying kmВІ gained/lost per period
- **Historical Trend Analysis** вҖ” 24-month ERA5 lookback with monthly aggregation, rendered as interactive Recharts graphs; registered regions now fetch ERA5 trends directly rather than relying on stale DB records

### рҹ§  Advanced Intelligence
- **ML vs GloFAS Comparison Panel** вҖ” side-by-side comparison of the ensemble model's prediction against live GloFAS discharge signal; includes agreement score, per-source explanation blocks (cyan for GloFAS, violet for ML), and a data snapshot of the exact inputs used вҖ” frontend passes the already-computed prediction so the explain endpoint never re-derives and diverges
- **Always-Present Explanation Reasons** вҖ” every explain response includes a quantitative data snapshot (rainfall 7d/30d, precip anomaly, soil moisture, elevation, GloFAS discharge ratio) so the reasons panel is never empty
- **Financial Impact Simulation** вҖ” translates flood depth and income class into Value-at-Risk estimates with tiered USD formatting (`$1.2M`, `$450K`, `$320`) and a confidence badge (HIGH / MEDIUM / LOW); no more `$0K` for sub-thousand values
- **AI NLG Summaries** вҖ” integrates with `gpt-4o-mini` to generate human-readable narrative insights from raw risk vectors
- **Compound Risk Engine** вҖ” models cascading hazard interactions (Flood вҶ’ Landslide вҶ’ Infrastructure failure)
- **Human-in-the-Loop Feedback** вҖ” analysts can verify or correct predictions, queuing data for the next re-training epoch

### рҹ—әпёҸ Navigation & UX
- **Satellite Base Map** вҖ” Esri World Imagery + Carto Dark Labels via MapLibre GL JS (`react-map-gl`)
- **Global Geocoding Search** вҖ” type any city, country, or coordinates (e.g. `26.0 85.5`) and fly there instantly
- **Ad-Hoc Location Analysis** вҖ” click anywhere on the map to generate an instant risk assessment
- **Role-Based Auth** вҖ” JWT-authenticated users with Admin / Analyst / Viewer access levels
- **Auto-Scheduler** вҖ” background monitoring jobs that re-evaluate all AOIs on a configurable interval

## рҹҡҖ Getting Started

### Prerequisites
- Python 3.10+
- Node.js 18+
- `pip` and `npm`

### 1. Clone the repo
```bash
git clone https://github.com/PranavAndhale/cosmeon.git
cd cosmeon
```

### 2. Backend Setup
```bash
# Create + activate virtualenv
python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install Python dependencies
pip install -r requirements.txt

# Seed the database with demo data
python seed_demo.py

# Train the ML model
python train_model.py

# Start the API server
uvicorn api.main:app --host 0.0.0.0 --port 8000 --reload
```

The API will be live at `http://localhost:8000`. Visit `http://localhost:8000/docs` for the interactive Swagger UI.

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

The app will be available at `http://localhost:3000`.

## рҹҢҗ Deployment

This project is deployed on **[Render](https://render.com)** using Docker.

- **Live URL**: [https://cosmeon.onrender.com](https://cosmeon.onrender.com)
- **Deploy Config**: [`render.yaml`](render.yaml)

Every push to `main` triggers an automatic redeploy on Render.

The **landing page** is also deployed as a standalone service at a separate Render URL via the [`cosmeon-landing`](https://github.com/PranavAndhale/cosmeon-landing)(wait for sometime if the render keeps cycling, it will load) repository вҖ” an Express static server serving the same Webflow-exported HTML.
